# &nbsp;Simulador de Teoria dos Conjuntos

# 

# Este projeto é uma aplicação interativa desenvolvida em Python que permite realizar operações matemáticas entre dois conjuntos: um definido manualmente pelo usuário e outro gerado de forma randômica pelo sistema.

# 

# 

# 

# Descrição do Problema

# 

# O programa foi concebido para manipular coleções de dados respeitando as propriedades fundamentais da Teoria dos Conjuntos. Ele resolve o desafio de validar entradas, garantir a unicidade dos elementos e processar interações lógicas.

# 

# Regras de Funcionamento:

# &nbsp;Cardinalidade: O sistema exige que ambos os conjuntos tenham entre 4 e 8 elementos.

# &nbsp;Tipos de Dados: São aceitos elementos alfanuméricos (letras de 'A-Z | a-z'- maiúscula ou minúscula, números de '0-9' em ambos os conjuntos e palavras no conjunto A). 

# &nbsp;    Justificativa: O uso de strings alfanuméricas permite simular desde conjuntos numéricos simples até conjuntos de identificadores complexos.

# &nbsp;Validação de Intervalo: O programa monitora em tempo real se o conjunto atingiu o tamanho mínimo ($MIN = 4$) para permitir a finalização ou se atingiu o máximo ($MAX = 8$) para interromper a entrada.

# 

# 

#
